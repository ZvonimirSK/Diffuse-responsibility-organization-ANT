# Diffuse Responsibility in a Small Organization  
*(ANT + Systems Theory Case Study — Work in Progress)*

## Overview
This project examines how responsibility becomes difficult to locate in a small organizational setting.

Rather than treating responsibility as an individual attribute, the analysis approaches the organization as a sociomaterial network in which decisions emerge through interactions between people, tools, and routines.

The study combines:
- Actor-Network Theory (ANT)
- Luhmann’s systems theory (decision communication, decision premises)

The case is anonymised and presented at an analytical level to foreground organizational mechanisms rather than identifiable actors or sites.

## Core Insight
Responsibility is not simply assigned or avoided.  
It is structurally produced — and often diffused — through:

- shared communication channels (e.g. inboxes)
- informal task distribution (“everyone does everything”)
- spreadsheets and documents without clear ownership
- routinized decision patterns

These elements function as mediators that shape how decisions are made and how accountability is distributed.

## Analytical Focus
The analysis focuses on:

- how irritations are translated into decision situations  
- how decision premises absorb uncertainty  
- how sociomaterial arrangements stabilize routines  
- how responsibility becomes diffused across the network  

## Key Concepts
- Decision communication (Luhmann)
- Decision premises (programmes, personnel, communication channels)
- Uncertainty absorption
- Deparadoxization
- Actor-network / mediators / obligatory passage points
